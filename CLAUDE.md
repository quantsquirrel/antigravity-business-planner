# Project Instructions

## 프로젝트 분석 리포트 동기화

스킬, 워크플로우, 템플릿, 규칙 등 프로젝트 구조에 변경이 생기면 반드시 아래 파일을 함께 업데이트합니다:

- **`output/reports/Antigravity-Business-Planner.html`**

### 업데이트 대상 섹션
- Hero 태그 (Workflows · Skills · Templates 숫자)
- 통계 카드 (워크플로우, 전문 스킬 숫자)
- 프로젝트 구조 트리 (디렉토리/파일 목록)
- 구조 분석 테이블 (rules/workflows/skills 개수)
- 워크플로우 테이블 (슬래시 명령어, 기능, 저장 위치)
- 스킬 테이블 (스킬명, 설명, 부가 스크립트)
- 템플릿 테이블 (템플릿명, 용도, 주요 섹션)
- 자동 세팅 타임라인 (단계별 개수)
- 추천 기획 프로세스 (Step 흐름)

### 규칙
- 숫자(개수)가 변경되면 모든 관련 위치를 일괄 업데이트합니다
- 새 항목 추가 시 해당 테이블에 행을 추가합니다
- 프로세스 흐름이 변경되면 타임라인 섹션을 수정합니다
