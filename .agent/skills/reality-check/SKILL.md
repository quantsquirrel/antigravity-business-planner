# reality-check

시장조사 AI 추정치를 실제 데이터와 대조 검증합니다. AI가 생성한 시장 규모, 경쟁사 정보, 가격 데이터 등을 공개 데이터 소스와 비교하여 신뢰도를 판정합니다.

## 활성화 조건
* /market-research 워크플로우 실행 시 자동 연동
* `output/research/` 폴더에 AI 추정치가 포함된 리서치 결과가 존재할 때

## 수행 작업

### AI 추정치 수집
* TAM/SAM/SOM 추정 수치를 리서치 결과에서 추출합니다
* 경쟁사 가격 및 시장 점유율 추정치를 수집합니다
* 성장률, 사용자 수 등 정량적 추정치를 목록화합니다

### 실데이터 소스 매칭
* 각 추정치에 대해 최적의 검증 소스를 자동 추천합니다
* 한국 시장 데이터는 네이버 데이터랩, 통계청을 우선 활용합니다
* 글로벌 데이터는 Google Trends, SimilarWeb 등을 활용합니다

### 비교 분석 실행
* 추정치와 실데이터의 차이율을 계산합니다
* 차이 원인을 분석하고 보정 방향을 제시합니다
* 검증 불가 항목은 명시적으로 "[미검증]"으로 표기합니다

### 신뢰도 판정
* 차이율 기반으로 신뢰도 등급을 업그레이드/유지합니다
* 최종 검증 리포트를 생성합니다

## 검증 소스 목록

| 소스 | 검증 대상 | 접근 방법 | 비용 |
|------|----------|----------|------|
| Google Trends | 검색 트렌드, 관심도 추이 | trends.google.com 검색 | 무료 |
| SimilarWeb | 웹사이트 트래픽, 경쟁사 방문자 수 | similarweb.com 조회 | 무료 (기본) |
| Crunchbase | 투자 라운드, 기업 가치평가 | crunchbase.com 검색 | 무료 (기본) |
| Product Hunt | 신규 론칭 데이터, 사용자 반응 | producthunt.com 검색 | 무료 |
| Reddit/커뮤니티 | 사용자 반응, 불만, 수요 시그널 | 서브레딧/커뮤니티 검색 | 무료 |
| App Store/Play Store | 앱 순위, 다운로드 수, 평점 | 스토어 직접 조회 | 무료 |
| 네이버 데이터랩 | 한국 검색 트렌드, 쇼핑 인사이트 | datalab.naver.com | 무료 |

## 대조 프로세스

### Step 1: AI 추정치 수집
* `output/research/` 내 리서치 결과에서 정량적 추정치를 추출합니다
* 추출 대상: TAM/SAM/SOM, 경쟁사 가격, 시장 성장률, 사용자 수
* 각 추정치의 원래 신뢰도 등급(보통 C)을 기록합니다

### Step 2: 실데이터 소스 선택
* 각 추정치 유형에 맞는 검증 소스를 자동 추천합니다
* 시장 규모 → Google Trends + SimilarWeb + 네이버 데이터랩
* 투자/기업 → Crunchbase
* 가격 → 경쟁사 실제 가격 페이지 직접 확인
* 사용자 반응 → Reddit, Product Hunt, App Store 리뷰

### Step 3: 실데이터 수집
* 추천된 소스에서 데이터를 검색합니다
* 브라우저 접근이 불가능한 경우 사용자에게 수동 입력을 요청합니다
* 수집된 데이터의 날짜와 출처를 반드시 기록합니다

### Step 4: 비교 분석
* AI 추정치와 실데이터를 항목별로 비교합니다
* 차이율을 계산합니다: `|추정치 - 실제| / 실제 × 100%`
* 차이 원인을 분석합니다 (데이터 시점 차이, 지역 차이, 정의 차이 등)

### Step 5: 신뢰도 업그레이드 판정
* 차이율 기반으로 신뢰도 등급을 결정합니다
* C→B 업그레이드 조건: 차이율 30% 이내
* 판정 결과를 원본 리서치 문서에 반영합니다

## 신뢰도 업그레이드 매트릭스

| 차이율 | 판정 | 신뢰도 변경 | 조치 |
|--------|------|------------|------|
| <30% | 검증됨 (Verified) | C→B | 신뢰도 업그레이드, 소스 URL 첨부 |
| 30-50% | 부분 검증 (Partial) | C 유지 | 주석 추가: "[부분 검증 — 차이율 X%]" |
| >50% | 재검토 필요 (Review) | C 유지 | 경고: "[재검토 필요 — 추정치 신뢰 불가]" |

## API 실패 시 Fallback

* 데이터 소스 접근 실패 시 신뢰도 C를 유지합니다
* 해당 항목에 "[미검증 — 수동 확인 필요]" 경고를 추가합니다
* 대체 소스가 있으면 자동으로 시도합니다
* 모든 소스 실패 시 사용자에게 수동 데이터 입력을 안내합니다

## 경쟁사 가격 크로스체크

경쟁사 가격은 AI 추정치와 실제 가격 사이 괴리가 큰 항목입니다. 반드시 실제 가격 페이지를 확인합니다.

### 확인 순서
1. 경쟁사 공식 웹사이트의 Pricing 페이지 직접 방문
2. 가격 티어별 기능 비교표 수집
3. 연간/월간 할인율 확인
4. 무료 플랜 제공 여부 및 제한사항 파악
5. AI 추정 가격과 실제 가격의 차이율 산출

### 주의사항
* 가격 페이지가 로그인 뒤에 숨겨진 경우 "[가격 비공개 — 문의 필요]"로 표기
* 엔터프라이즈 가격은 "커스텀"으로 분류하고 추정치에서 제외
* 환율 변동 고려: 수집 시점의 환율을 명시

```markdown
## 항목별 상세

```
