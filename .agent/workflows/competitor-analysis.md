# competitor-analysis

경쟁사를 체계적으로 분석합니다.

## 분석 항목
* 직접 경쟁사 3-5개를 식별합니다
* 간접 경쟁사 및 대체재를 파악합니다
* Porter's Five Forces 분석을 수행합니다
* 각 경쟁사별 SWOT 매트릭스를 작성합니다
* 경쟁사의 가격 전략, 마케팅 방식, 핵심 강점을 비교합니다
* 시장 내 포지셔닝 맵을 그립니다
* 우리 사업의 차별화 포인트를 도출합니다

## 출력 형식
* 경쟁사 비교표를 작성합니다
* 포지셔닝 맵을 텍스트 기반으로 시각화합니다
* output/research/ 폴더에 결과를 저장합니다

## AI 경쟁 분석 (v2.0 Phase 6)

idea.json에 `ai_business.detected: true`가 설정된 AI 사업의 경우, 아래 AI 특화 경쟁 분석을 추가로 수행합니다.

### AI 기술 스택 비교
* 각 경쟁사의 AI 기술 스택을 분류합니다: 자체 모델 vs API 의존 vs 하이브리드
* 사용 중인 LLM, 임베딩 모델, 벡터DB를 비교합니다
* AI 파이프라인 성숙도를 평가합니다 (RAG, 에이전트, 파인튜닝 등)

### 데이터 경쟁력 비교

| 경쟁사 | 데이터 소스 | 독점 데이터 | 데이터 축적 구조 | 데이터 경쟁력 |
|--------|-----------|-----------|----------------|-------------|
| | | 있음/없음 | | 강/중/약 |

### AI 비용 구조 비교
* 경쟁사의 추정 AI 비용 구조를 분석합니다 (공개 정보 기반)
* 가격 대비 AI 비용 비중을 추정합니다
* 비용 효율성에서의 우위/열위를 판단합니다

### 기술 방어력 평가

| 경쟁사 | 복제 난이도 | 데이터 해자 | 네트워크 효과 | 기술 방어력 |
|--------|-----------|-----------|-------------|-----------|
| | 1주/1-3개월/3-6개월/6개월+ | 있음/없음 | 있음/없음 | 강/중/약 |

### 출력 형식
* AI 기술 스택 비교표를 작성합니다
* 데이터 경쟁력 매트릭스를 작성합니다
* output/research/ 폴더에 기존 경쟁 분석과 함께 저장합니다
