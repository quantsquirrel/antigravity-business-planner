# legal-checklist

사업에 필요한 법률 및 인허가 사항을 체계적으로 점검합니다.

> **면책 조항**: 이 정보는 일반적인 안내이며, 법률 전문가의 자문을 받으시길 권합니다.

## 체크리스트 항목

* 사업자 등록 절차 (개인/법인 비교, 과세 유형 선택)
* 업종별 필요 인허가 목록
* 위생 및 안전 관련 규정 (식품위생법, 소방법 등)
* 근로기준법 관련 사항 (고용 계약, 4대 보험, 최저임금)
* 세금 관련 사항 (부가세, 종합소득세, 법인세)
* 지적재산권 보호 (상표 등록, 특허)
* 필요 보험 가입 사항
* 임대차 계약 시 주의사항
* 프랜차이즈인 경우 가맹사업법 관련 사항

## 1. 사업자 등록 절차

### 개인사업자 vs 법인 비교

| 구분 | 개인사업자 | 법인(주식회사) |
|------|-----------|--------------|
| 설립 비용 | 무료 (등록면허세 정도) | 50~100만원 (등록면허세, 법무사 비용 등) |
| 설립 기간 | 1~2일 | 7~14일 |
| 세율 | 종합소득세 6~45% (누진) | 법인세 9~24% + 배당소득세 |
| 책임 범위 | 무한 책임 (개인 재산 포함) | 유한 책임 (출자 범위) |
| 대외 신뢰도 | 보통 | 높음 (B2B, 투자 유치에 유리) |
| 자금 인출 | 자유 | 급여/배당 절차 필요 |
| 4대 보험 | 대표 가입 의무 없음 | 대표 가입 의무 |
| 추천 케이스 | 1인 창업, 초기 테스트, 연매출 5천만원 이하 | 투자 유치, B2B, 연매출 1억원 이상 예상 |

> **관할 기관**: 국세청 (홈택스), 등기소 (법인 설립 시)

### 간이과세자 vs 일반과세자

| 구분 | 간이과세자 | 일반과세자 |
|------|-----------|-----------|
| 기준 | 연 매출 8,000만원 미만 | 연 매출 8,000만원 이상 또는 선택 |
| 부가세율 | 업종별 1.5~4% (부가가치율 적용) | 10% (매입세액 공제 가능) |
| 부가세 신고 | 연 1회 (1월) | 연 2회 (1월, 7월) |
| 세금계산서 발행 | 연 매출 4,800만원 이상 시 가능 | 의무 발행 |
| 매입세액 공제 | 제한적 | 전액 공제 |
| 추천 케이스 | 소규모, B2C 중심 | B2B, 초기 투자 비용 큰 경우 |

> 간이과세 배제 업종(부동산임대, 과세유흥장소 등)은 매출과 무관하게 일반과세 적용

### 업종코드(KSIC) 선택 가이드

| 항목 | 내용 |
|------|------|
| 주업종 | 매출 비중이 가장 큰 사업 영역 1개 선택 |
| 부업종 | 추가 사업 영역 (최대 5개까지 등록 가능) |
| 변경 시 주의 | 업종 변경 시 인허가 재확인 필요, 간이과세 배제 업종 여부 확인 |
| 참고 사이트 | 통계청 KSIC 검색 (kostat.go.kr) |

> **팁**: 업종코드에 따라 세제 혜택, 정부 지원사업 자격이 달라지므로 신중하게 선택하세요.

### 사업자등록 필요 서류 체크리스트

| ☐ | 서류 | 비고 |
|---|------|------|
| ☐ | 사업자등록 신청서 | 홈택스에서 다운로드 또는 온라인 작성 |
| ☐ | 대표자 신분증 사본 | 주민등록증 또는 운전면허증 |
| ☐ | 사업장 임대차계약서 사본 | 자가 소유 시 건물등기부등본 |
| ☐ | 업종별 인허가 증빙 | 해당 업종에 한함 (식품영업허가 등) |
| ☐ | 동업계약서 | 공동사업자인 경우 |
| ☐ | 법인등기부등본 | 법인사업자인 경우 |

### 홈택스 온라인 등록 절차

1. 홈택스(hometax.go.kr) 접속 → 회원가입/로그인
2. [신청/제출] → [사업자등록 신청(개인)]
3. 인적사항, 사업장 정보, 업종코드 입력
4. 구비서류 스캔 첨부 업로드
5. 신청 완료 → 1~3 영업일 내 처리
6. 사업자등록증 PDF 다운로드

> **관할 기관**: 국세청 홈택스 (hometax.go.kr) | **관련 법규**: 부가가치세법 제8조

* 계약 기간, 갱신/해지 조건, 위약금을 명확히 합니다
* 지적재산권 귀속 조항을 반드시 포함합니다
* 비밀유지 조항(NDA)을 고려합니다
* 불가항력(Force Majeure) 조항을 포함합니다

## 5. AI 사업 추가 법률 사항 (v2.0 Phase 6)

idea.json에 `ai_business.detected: true`가 설정된 경우, AI 사업 특화 법률 사항을 추가로 점검합니다.

### 트리거 조건
* `output/ideas/{id}-{name}/idea.json`의 `ai_business.detected` 값이 `true`일 때 자동 활성화

### AI 기본법 대응

| 항목 | 내용 | 상태 |
|------|------|------|
| 고위험 AI 분류 | 의료, 금융, 채용, 사법 분야 AI는 고위험으로 분류 | 2025~ 시행 |
| 사전영향평가 | 고위험 AI 시스템은 출시 전 영향평가 의무 | 고위험 해당 시 |
| 투명성 의무 | AI 생성 콘텐츠임을 이용자에게 고지 | 전체 AI 서비스 |
| AI 윤리 기준 | 공정성, 책임성, 투명성, 안전성 원칙 준수 | 자율 준수 권고 |

| 쟁점 | 현행 해석 | 사업 대응 |

## 6. Micro-SaaS/소규모 사업 법률 (v2.0 Phase 7)

idea.json의 `business_scale`이 `"micro"` 또는 `"small"`인 경우, 간소화된 법률 체크리스트를 우선 적용합니다.

### 트리거 조건
* `output/ideas/{id}-{name}/idea.json`의 `business_scale` 값이 `"micro"` 또는 `"small"`일 때 자동 활성화
* 미설정 시 기존 전체 법률 체크리스트 프로세스 유지

### 1인 사업자 필수 최소 법률 체크리스트

| ☐ | 항목 | 우선순위 | 예상 비용 | 소요 시간 |
|---|------|---------|----------|----------|
| ☐ | 개인사업자 등록 (간이과세) | 필수 | 무료 | 1~2일 |
| ☐ | 통신판매업 신고 | 필수 (온라인 판매 시) | 무료 | 3~5일 |
| ☐ | 개인정보처리방침 게시 | 필수 | 무료 (직접 작성) | 1일 |
| ☐ | 서비스 이용약관 게시 | 필수 | 무료 (직접 작성) | 1일 |
| ☐ | 사업용 계좌 개설 | 권장 | 무료 | 1일 |
| ☐ | 도메인/상호 상표 검색 | 권장 | 무료 (검색), 21만원~(출원) | 1일~6개월 |

> 위 6개 항목만 완료하면 합법적으로 온라인 서비스 운영이 가능합니다.

### 통신판매업 신고
