# market-research

시장 조사를 체계적으로 수행합니다.

## 조사 항목
* 목표 시장의 전체 규모를 추정합니다 (TAM: Total Addressable Market)
* 실제 접근 가능한 시장 규모를 산출합니다 (SAM: Serviceable Addressable Market)
* 초기 목표 시장 점유율을 설정합니다 (SOM: Serviceable Obtainable Market)
* 최근 3-5년간 산업 트렌드를 분석합니다
* 주요 고객 세그먼트를 정의하고 각 세그먼트의 특성을 파악합니다
* 시장 성장률과 향후 전망을 예측합니다
* 시장 진입 장벽을 분석합니다

## 출력 형식
* 각 항목을 표와 그래프로 정리합니다
* 데이터 출처를 반드시 명시합니다
* output/research/ 폴더에 결과를 저장합니다

## AI 시장 조사 보충 (v2.0 Phase 6)

AI 사업 아이디어가 감지된 경우, 기존 시장 조사 항목에 아래 내용을 추가로 조사합니다.

### AI 기술 시장 규모

| 시장 | 글로벌 규모 (2025 추정) | 국내 규모 (2025 추정) | 성장률 (CAGR) |
|------|----------------------|---------------------|-------------|
| AI 전체 | $190B+ | 5조원+ | 35~40% |
| 생성형 AI | $60B+ | 1.5조원+ | 40~50% |
| AI SaaS | $30B+ | 8,000억원+ | 25~30% |
| AI 인프라 (MLOps, 벡터DB 등) | $15B+ | 3,000억원+ | 30~35% |
| AI 에이전트 | $5B+ | 1,000억원+ | 50~60% |

> 수치는 2025년 기준 추정치이며, MCP 연동 시 최신 데이터로 대체됩니다. 미연동 시 "[AI 추정]" 표기.

### AI 규제 환경

| 규제 | 상태 | 핵심 내용 | 사업 영향 |
|------|------|----------|----------|
| 개인정보보호법 | 시행 중 | AI 학습 데이터의 개인정보 처리 동의, 가명처리 규정 | 데이터 수집/활용 범위 제한 |
| AI 기본법 | 2025~ 시행 | 고위험 AI 사전영향평가, 투명성 의무, AI 윤리 기준 | 고위험 분야(의료/금융/채용) 진입 장벽 |
| 저작권법 (AI 관련) | 논의 중 | AI 학습 데이터 저작권, AI 생성물 권리 귀속 | AI 콘텐츠 사업 리스크 |

## Micro-SaaS/소규모 시장 조사 (v2.0 Phase 7)

idea.json의 `business_scale`이 `"micro"` 또는 `"small"`인 경우, 기존 시장 조사 항목에 아래 내용을 추가하거나 대체하여 조사합니다.

### 트리거 조건
* `output/ideas/{id}-{name}/idea.json`의 `business_scale` 값이 `"micro"` 또는 `"small"`일 때 자동 활성화
* 미설정 시 기존 TAM/SAM/SOM 기반 프로세스 유지

### 니치 시장 분석 (TAM/SAM/SOM 대체)

기존 TAM/SAM/SOM 대신 아래 프레임워크를 사용합니다:

| 분석 항목 | 설명 | 출력 형식 |
|----------|------|----------|
| 니치 시장 규모 | 타겟 커뮤니티/포럼 규모, 월간 활성 사용자 추정 | 표 |
| ICP (Ideal Customer Profile) | 핵심 고객 1명의 상세 프로필 (직업, 불편, 지불 의향, 현재 대안) | 카드 |
| 예상 전환율 | 유사 니치 SaaS 벤치마크 (일반 2-5%, 고니치 5-15%) | 수치 |
| 커뮤니티 활성도 | Reddit, 디스코드, 네이버 카페, 포럼 활동량 분석 | 표 |
| 키워드 수요 | 핵심 키워드 월간 검색량, 경쟁 난이도 | 표 |
| 기존 솔루션 불만 | 경쟁 제품 리뷰에서 반복되는 불만 패턴 | 목록 |

### 니치 검증 체크리스트

| 항목 | 기준 | 판정 |
|------|------|------|
| 커뮤니티 규모 | 1,000명+ 활성 멤버 | ✅ Pass / ❌ Fail |
| "불만" 시그널 | 기존 솔루션 리뷰 ≤3.5점 또는 반복 불만 3개+ | ✅ / ❌ |
| 지불 의향 시그널 | "I'd pay for...", "무료면 좋겠다" 등 5건+ | ✅ / ❌ |
| 경쟁사 수 | 직접 경쟁 ≤5개 (과포화 아닌지) | ✅ / ❌ |
| 키워드 검색량 | 월 1,000+ 검색 (영어) 또는 500+ (한국어) | ✅ / ❌ |

> 5개 중 3개 이상 Pass 시 "니치 유효", 2개 이하 시 "니치 재검토" 판정

### 글로벌 vs 로컬 판단

| 판단 기준 | 글로벌 추천 | 로컬 추천 |
|----------|-----------|----------|
| 언어 의존도 | 낮음 (도구, API) | 높음 (콘텐츠, 상담) |
| 규제 영향 | 낮음 | 높음 (인허가, 자격) |
| 결제 인프라 | Stripe 사용 가능 | 로컬 PG 필요 |
| 마케팅 채널 | Reddit, Twitter, PH | 네이버, 카카오, 커뮤니티 |

### 출력 형식
* 기존 시장조사와 동일하게 마크다운 표/목록 형식
* output/research/ 폴더에 결과를 저장합니다
* 니치 검증 체크리스트 결과를 반드시 포함합니다
