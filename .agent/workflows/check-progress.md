# check-progress

사업 기획 진행 상황을 확인합니다.

## 수행 작업
* scripts/check_progress.py를 실행하여 현재 진행률을 확인합니다
* 아이디어 발굴(Stage 0) + 8단계 기획 프로세스 중 완료된 단계와 미완료 단계를 표시합니다
* Stage 0은 output/ideas/에 파일이 있을 때만 조건부로 표시됩니다
* 다음으로 진행해야 할 단계를 추천합니다
* 각 단계에서 사용할 워크플로우 명령어를 안내합니다

## 멀티 아이디어 모드
* `output/ideas/` 하위에 `idea.json`이 있는 아이디어 폴더가 있으면 자동으로 포트폴리오 모드로 전환됩니다
* `--portfolio` 옵션으로 전체 아이디어 포트폴리오 대시보드를 표시합니다
* `--idea {id}` 옵션으로 특정 아이디어의 진행률만 확인합니다
* 포트폴리오 모드 시 `output/ideas/portfolio.md`를 자동 갱신합니다

## 출력 형식
* 각 단계를 ✅/⬜ 로 표시합니다
* 전체 진행률을 % 와 프로그레스 바로 보여줍니다
* 추천 다음 액션을 제시합니다
