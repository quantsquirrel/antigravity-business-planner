# Data Governance Checklist

AI 서비스의 개인정보 보호와 데이터 관리 가이드라인입니다.

> "PII"란 이름, 전화번호, 이메일 같은 **개인을 식별할 수 있는 정보**를 말합니다.
> "익명화"란 **누구인지 알 수 없게 정보를 변환하는 것**입니다.
> "GDPR"이란 **유럽 개인정보보호법** (해외 서비스 시 적용)입니다.

---

## 기본 규칙

* 프롬프트 로그에 개인정보(PII)가 포함되는 경우 익명화 처리를 필수로 합니다.
* PII 마스킹 계획을 모든 AI 서비스 산출물에 필수 포함합니다.
* 사용자 데이터를 수집할 때는 반드시 목적을 명시하고 동의를 받습니다.
* 수집한 데이터는 명시된 목적 외에 사용하지 않습니다.
* 데이터 보관 기간을 정하고, 기간이 지나면 삭제합니다.

---

## PII 마스킹 계획

모든 AI 서비스 산출물에 아래 항목을 포함하세요.

### 마스킹 대상 정보

| 정보 유형 | 예시 | 마스킹 방법 |
|----------|------|-----------|
| 이름 | 홍길동 | 해시 처리 또는 "사용자A" 치환 |
| 이메일 | user@email.com | h***@e***.com 형태 |
| 전화번호 | 010-1234-5678 | 010-****-5678 |
| 주소 | 서울시 강남구 ... | 시/구 단위까지만 |
| 카드번호 | 1234-5678-... | ****-****-****-5678 |
| 주민등록번호 | 절대 수집/저장 금지 | - |

### 마스킹 적용 시점

- [ ] AI 모델에 데이터를 전달하기 전
- [ ] 로그에 기록하기 전
- [ ] 분석 리포트를 생성하기 전
- [ ] 외부 서비스(API)에 데이터를 보내기 전

## AI 기본법 기본 체크리스트

한국의 AI 기본법(2026년 시행 예정) 기준 사전 점검 항목입니다.

### 고위험 AI 해당 여부 확인

- [ ] 의료 분야 AI인가? (진단, 처방 보조)
- [ ] 금융 분야 AI인가? (신용평가, 대출 심사)
- [ ] 채용 분야 AI인가? (이력서 선별, 면접 평가)
- [ ] 교육 분야 AI인가? (성적 평가, 입학 심사)
- [ ] 법 집행 분야 AI인가? (범죄 예측, 감시)

> 위 분야에 해당하면 "고위험 AI"로 분류되며, 추가 의무가 적용됩니다.

### 투명성 고지

- [ ] 사용자에게 AI가 사용되고 있다는 사실을 고지하고 있는가?
- [ ] AI의 판단이 최종 결정에 어떤 영향을 미치는지 설명하고 있는가?
- [ ] AI 생성 콘텐츠에 "AI가 생성한 콘텐츠입니다" 표시가 있는가?

### 이의제기 절차

- [ ] 사용자가 AI의 판단에 이의를 제기할 수 있는 창구가 있는가?
- [ ] 이의 제기 시 사람이 재검토하는 절차가 있는가?
- [ ] 이의제기 처리 기한을 안내하고 있는가? (권장: 14일 이내)

## GDPR 기본 체크리스트

해외(특히 유럽) 사용자에게 서비스할 때 적용됩니다.

### 동의 획득 절차

- [ ] 개인정보 수집 전 명시적 동의를 받고 있는가?
- [ ] 동의 시 수집 항목, 목적, 보관 기간을 안내하는가?
- [ ] 동의를 거부해도 기본 서비스를 이용할 수 있는가?
- [ ] 마케팅 동의는 별도로 받고 있는가?

### 데이터 삭제 요청 처리 ("잊힐 권리")

- [ ] 사용자가 자신의 데이터 삭제를 요청할 수 있는 방법이 있는가?
- [ ] 삭제 요청 처리 기한을 정해두었는가? (GDPR 기준: 30일 이내)
- [ ] 삭제 시 백업 데이터도 함께 삭제하는가?
- [ ] 삭제 완료를 사용자에게 알리는가?

### 데이터 국외이전 고지

- [ ] 데이터가 어느 국가에 저장되는지 명시하고 있는가?
- [ ] 유럽 외 국가로 데이터 이전 시 적절한 보호 조치가 있는가?
- [ ] 서드파티(외부 서비스)에 데이터 전달 시 사전 고지하는가?

## 도메인별 강화 모듈

서비스 도메인에 따라 추가 체크리스트가 활성화됩니다.

### 조건별 분기

| 조건 | 동작 |
|------|------|
| 일반 도메인 (쇼핑몰, 생산성 도구 등) | 위 기본 체크리스트만 적용 |
| 의료 도메인 | 기본 + HIPAA 체크리스트 추가 |
| 금융 도메인 | 기본 + KYC/AML 체크리스트 추가 |

### 의료 도메인: HIPAA 체크리스트

HIPAA(Health Insurance Portability and Accountability Act)는 미국 의료 정보 보호법입니다. 의료 데이터를 다루는 서비스에 적용됩니다.

- [ ] PHI(보호 대상 건강 정보)를 암호화하여 저장하는가?
- [ ] 의료 데이터 접근 로그를 기록하는가?
- [ ] BAA(Business Associate Agreement)를 체결했는가?
- [ ] 의료 데이터 유출 시 72시간 이내 통지 절차가 있는가?
- [ ] 직원/외주의 HIPAA 교육 기록이 있는가?

### 금융 도메인: KYC/AML 체크리스트

KYC(Know Your Customer, 고객 확인)와 AML(Anti-Money Laundering, 자금세탁방지)은 금융 서비스 필수 요건입니다.

- [ ] 고객 본인 확인(신분증 검증) 절차가 있는가?
- [ ] 의심 거래 모니터링 시스템이 있는가?
- [ ] 거래 기록을 법정 기간(5년) 이상 보관하는가?
- [ ] 자금세탁 방지 담당자를 지정했는가?
- [ ] 정기 교육(연 1회 이상)을 실시하고 있는가?
