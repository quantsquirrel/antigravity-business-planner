## 3. 파일 구조

> 아래는 Next.js App Router 기준 예시입니다. 선택한 기술 스택에 맞게 수정하세요.
> 잘 모르겠으면 이 섹션을 비워두세요 — AI가 자동 제안합니다.

```
프로젝트루트/
├── app/
│   ├── layout.tsx          # 전체 레이아웃
│   ├── page.tsx            # 메인 페이지
│   ├── (auth)/
│   │   ├── login/page.tsx  # 로그인
│   │   └── signup/page.tsx # 회원가입
│   └── dashboard/
│       └── page.tsx        # 대시보드
├── components/
│   ├── ui/                 # 공통 UI 컴포넌트
│   └── features/           # 기능별 컴포넌트
├── lib/
│   ├── db.ts               # DB 연결
│   └── auth.ts             # 인증 설정
├── public/                 # 정적 파일 (이미지 등)
└── .env.local              # 환경 변수 (비밀키)
```
